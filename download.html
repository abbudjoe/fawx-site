Quick start

Up and running in three steps.

Install

Open the DMG. Drag Fawx to Applications.

Setup

Launch Fawx. The setup wizard connects your AI provider: Claude, ChatGPT, or your own API key. Credentials are encrypted locally with AES-256-GCM.

Go

Start chatting. Fawx handles tool calling, memory, and multi-step reasoning automatically.

Prefer the terminal?

Fawx includes a full terminal UI alongside the native app.

$ fawx tui
